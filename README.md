# YOLOv8 小蚂蚁检测项目

使用 Ultralytics YOLOv8 训练蚂蚁（小物体）检测模型，mAP@0.5: 0.98，mAP@0.5:0.95: 0.64

。包含 Gradio demo。

## 文件
- best.pt: 训练模型
- best.onnx: 导出 ONNX 模型
- yolov8_ant_detection.ipynb: 完整代码

## Demo
[https://36132589cd80c296aa.gradio.live] 

## 技能
- YOLOv8 自定义训练
- 小物体检测优化
- Gradio 部署

## 与研究方向
聚焦小物体感知，适用于移动视觉系统。
